- Hay dos tipos de formas para conectar los cables trenzados

- La Tipo A o la Tipo B

- Nosotros empleamos la forma tipo B

- Utilizamos el par trenzados de los colores naranja para la conexión positiva, lo conectamos a +V de la fuente que sera el positivo.

- Utilizamos el par trenzados de los colores verde para la conexión negativa, lo conectamos a -V de la fuenteque sera el negativo.

- Utilizamos el color naranja para la conexión positiva.

- Utilizamos el color naranja/blanco para la conexión negativa.

- Utilizamos el color naranja para la conexión positiva.

- Utilizamos el color naranja/blanco para la conexión negativa.

- Utilizamos el par trenzados de los colores naranja para la conexión positiva.
- Utilizamos el par trenzados de los colores verde para la conexión negativa.

- Utilizamos el color azul para la conexión positiva.

- Utilizamos el color azul/blanco para la conexión negativa.

- En la entrada ETHENDER conectaremos el cable de red que viene desde el moden o proveedor de internet ej fibertel.

- En la entrada VGA o HDMI conectaremos la pantalla en la cual se podra ver lo que graban las camaras.
